# Laravel E Commerce App

## Install

`git clone https://github.com/ozgurkaracam/laravelapps.git`

`cd ecommerce`

`npm install`

`cp .env.example .env`

prepare env.

`php artisan key:generate`

`php artisan migrate --seed`

`php run serve`
